Steps to configure this Flask Application on a new device:-

1) Install Python

2) Use "pip install -r requirements.txt" to recursively install everything in requirements.txt

3) Use flask run --host=0.0.0.0 to run flask on the local server

4) The credentials for bank are "BANK" and "bank@gmail.com"

ALTERNATIVELY CLICK setup.bat (It automates steps 1-5)
